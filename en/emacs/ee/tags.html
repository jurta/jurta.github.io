ee-tags - etags facility

Description

This extension is the replacement for etags user interface. It is more usable and flexible than original etags listings. But ee-tags still uses some etags functions.

Default views

By File Name/Sorted by Tag

This view displays the alphabetically sorted list of tags grouped by file names.

c-path-finder groups by file name
r-sorter sorts alphabetically by tag name
r-printer prints the tag name
r-select switches to the tag location
