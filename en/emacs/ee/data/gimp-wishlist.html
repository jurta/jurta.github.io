gimp-wishlist.ee - GIMP wishlist

Data Description

This data file holds a list of GIMP wishes extracted from http://jerhard.org/gimp-wishes-lisp by Emacs-Lisp program gimp-wishlist.el to ee data file gimp-wishlist.ee.

Useful Views

Views are described in the view file view/gimp-wishlist.ee.

by Priority/by Category

This view displays a list of GIMP wishes, grouped by priority and category.

c-path-finder groups by priority and category
r-printer prints wish name
r-select prints full information
