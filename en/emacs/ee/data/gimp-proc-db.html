gimp-proc-db.ee - GIMP procedural database

Data Description

This data file holds a list of GIMP procedures extracted from output of GIMP's function 'gimp-procedural-db-dump' by Emacs-Lisp program gimp-proc-db.el to ee data file gimp-proc-db.ee.

Useful Views

Views are described in the view file view/gimp-proc-db.ee.

This view displays a list of GIMP procedures, grouped by procedure type.
