--- translations: en: /en/emacs/ee title: Ее - Менеджер Информации для Емакса ---

Ее - категорийный менеджер информации для Емакса, который преобразует информацию из разных источников в реляционную или ассоциативную форму, после чего использует поля данных для построения деревьев категорий, которые отображаются в буферах Емакса с возможностью произведения пользователем различных операций над ними. Правила для этого задаются аналогичными структурами данных, и включают в себя правила для построения путей категорий для добавления в дерево категорий, сортировки данных и категорий, фильтрации исходных записей, вычислений на полях данных и категориях, вывода дерева категорий и данных.

Более подробная информация доступна на английском.
