ee-windows - display and switch Emacs window configurations

Description

The command `ee-windows-add' adds the current window configuration to the global list of window configurations. Then the command `ee-windows' can be called to display and switch the added window configurations.

Default views

By Name

This view presents a list of saved window configurations, where every window configuration is displayed by the comma-separated list of buffer names of the buffers which were displayed on the current frame before the saving of the window configuration.

r-printer comma-separated list of window buffer names
r-select switches to the saved window configuration
