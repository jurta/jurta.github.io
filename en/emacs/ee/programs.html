ee-programs - categorized program menu

Description

This extension builds a categorized menu of programs collected from operating system menu. Currently it supports Debian GNU/Linux menu system and M$Window$ Start-menu programs tree.

Default views

By Category

Programs in this view are categorized under the same categories as they have in the system menu.

c-path-finder creates the same category tree as in system menu
r-select calls the system command associated with menu item
