view/links.ee - list of links (bookmarks)

Data Description

Links data file holds a list of links (bookmarks). Examples of links data files:

ee.ee index of available ee extensions
ell.ee Emacs Lisp List

Useful Views

By Category

This view displays a list of links (bookmarks), grouped by their categories. After typing `RET' the selected link is opened in the Web browser. Views are stored in the view file view/links.ee and have the following properties:

r-printer prints link title and description
r-select opens selected links in the Web browser

By Domain Name

This view displays a list of links (bookmarks), categorized by domain name hierarchy, i.e. parts of domain name in the reverse order.

r-printer prints link title and description
r-select opens selected links in the Web browser

By Contact Name

This view displays a list of links (bookmarks), categorized by first letter of contact (author) name and the name itself.

r-printer prints link title and description
r-select opens selected links in the Web browser
