ee-gnus - summary and topic mode for Gnus

Description

This extension provides the integration with gnus package. It extracts group names from gnus internals and organizes them under different categories. This extension can replace the gnus topic mode.

Also ee could be used to implement summary mode for articles, where data from articles (author, date, subject, etc.) will be used as categories in the ee views.

Default views

By Category

This view categorizes gnus groups by their topic names.

c-path-finder hierarchical categories extracted from 'gnus-topic-topology'
c-sorter categories are sorted by sorting order of topics in the Gnus
r-sorter groups are sorted by their original sorting order in the Gnus
r-counter counts groups and postings in every group
r-printer prints the number of unread articles and group name
